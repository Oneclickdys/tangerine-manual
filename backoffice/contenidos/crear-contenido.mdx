## Descripción

La creación de contenidos permite a los administradores y gestores de contenido agregar nuevos recursos educativos al sistema. Tangerine soporta una amplia variedad de tipos de contenido: videos, audios, documentos PDF, presentaciones, actividades interactivas, enlaces externos y más.

<Info>
  **Rol requerido:** Administrador de Backoffice, Gestor de Contenido, Publisher
</Info>

<Note>
  Los usuarios con rol **Help Desk** no tienen acceso a la creación de contenidos.
</Note>

## Cómo acceder

<Steps>
  <Step title="Inicia sesion en el Backoffice">
    Accede al panel de administración con tus credenciales.
  </Step>
  <Step title="Navega a Contenidos">
    En el menú lateral izquierdo, haz clic en el icono de **Contenidos**.
  </Step>
  <Step title="Haz clic en Crear contenido">
    En la esquina superior derecha de la pantalla, encontrarás el botón **Crear contenido**.

    ![Botón Crear contenido](/assets/screenshots/backoffice/contenidos/crear-contenido-paso1-detalles.png)
  </Step>
</Steps>

---

## Estructura del formulario

El formulario de creación de contenidos se organiza en **cuatro pasos** que te guiaran a través de toda la configuración:

| Paso | Nombre | Descripción |
|------|--------|-------------|
| 1 | **Detalles** | Información básica del contenido y archivo adjunto |
| 2 | **Clasificación** | Metadatos educativos (nivel, asignatura, idioma, tags) |
| 3 | **Apariencia** | Miniatura (thumbnail) y configuración de privacidad |
| 4 | **Agrupacion** | Asociacion a publicaciones, colecciones y dependencias |

<Tip>
  Durante la creación inicial, solo el paso **Detalles** esta disponible. Los pasos adicionales (Clasificación, Apariencia, Agrupacion) se habilitan automáticamente después de guardar el contenido por primera vez.
</Tip>

---

## Paso 1: Detalles

Este es el paso principal donde defines la información básica del contenido y adjuntas el archivo o URL.

### Campos del formulario

| Campo | Requerido | Descripción |
|-------|-----------|-------------|
| **Nombre del contenido** | Sí | Título descriptivo del recurso |
| **Tipo de contenido** | Sí | Formato del recurso (Video, Audio, PDF, Link, etc.) |
| **Disponible para** | Sí | Define quien puede ver el contenido |
| **ERP** | No | Código identificador externo |
| **Descripción** | No | Descripción breve del contenido |
| **Estado** | No | Activo/Inactivo - Los contenidos inactivos no son visibles |
| **Accesible offline** | No | Permite descargar el contenido para uso sin conexión |

### Seleccionar tipo de contenido

El tipo de contenido determina que opciones adicionales estarán disponibles y que formatos de archivo se aceptan.

![Selector de tipos de contenido](/assets/screenshots/backoffice/contenidos/crear-contenido-tipos-contenido.png)

<AccordionGroup>
  <Accordion title="Tipos de contenido con archivo">
    Estos tipos requieren subir un archivo desde tu computadora:

    | Tipo | Formatos aceptados |
    |------|-------------------|
    | **Video** | zip, mp4, ogg |
    | **Audio** | mp3, ogg, wav |
    | **PDF** | pdf |
    | **Imagen** | jpg, png, gif, svg |
    | **OFFICE** | doc, docx, ppt, pptx, xls, xlsx |
    | **SCORM** | zip |
    | **ZIP** | zip |
    | **Tin-can xApi** | zip |
    | **HTML Interactivo** | zip |
    | **EPUB** | epub |
    | **App** | apk, ipa |
  </Accordion>
  <Accordion title="Tipos de contenido con URL">
    Estos tipos requieren introducir una URL externa:

    | Tipo | Descripción |
    |------|-------------|
    | **Link** | Enlace a cualquier recurso web externo |
    | **Video lesson** | Video de plataformas externas (YouTube, Vimeo) |
    | **LTI** | Herramientas externas via protocolo LTI |
  </Accordion>
  <Accordion title="Tipos de contenido con editor">
    Estos tipos se crean usando editores especializados de Tangerine:

    | Tipo | Descripción |
    |------|-------------|
    | **Question (Lemonade)** | Actividades interactivas creadas con el editor Lemonade |
    | **Video lesson** | Videolecciones con el editor de segmentos |
    | **Mint** | Paginas creadas con el editor Mint |
  </Accordion>
</AccordionGroup>

### Subir archivo de contenido

Cuando seleccionas un tipo de contenido que requiere archivo, aparece la sección **Adjuntar contenido**:

![Subir archivo de video](/assets/screenshots/backoffice/contenidos/crear-contenido-tipo-video.png)

<Steps>
  <Step title="Selecciona el tipo de contenido">
    Elige el tipo adecuado del selector (ej. Video, PDF, Audio).
  </Step>
  <Step title="Haz clic en el área de carga">
    Haz clic en **Seleccionar fichero** o arrastra el archivo al área indicada.
  </Step>
  <Step title="Espera la subida">
    Una barra de progreso mostrará el avance de la carga. No cierres la ventana durante este proceso.
  </Step>
</Steps>

<Warning>
  **Importante:** El archivo se sube al crear el contenido. Asegúrate de tener una conexión estable, especialmente para archivos grandes como videos.
</Warning>

### Introducir URL externa

Para contenidos de tipo **Link**, aparece un campo para introducir la URL:

![Contenido tipo Link](/assets/screenshots/backoffice/contenidos/crear-contenido-tipo-link.png)

<Steps>
  <Step title="Selecciona tipo Link">
    En el selector de tipo de contenido, elige **Link**.
  </Step>
  <Step title="Introduce la URL">
    Escribe o pega la URL completa del recurso externo (debe comenzar con http:// o https://).
  </Step>
</Steps>

### Opciones de disponibilidad

El campo **Disponible para** determina quien puede acceder al contenido:

| Opción | Descripción |
|--------|-------------|
| **Docentes y estudiantes** | Visible para todos los usuarios |
| **Solo docentes** | Solo visible para docentes (material de apoyo, guias del profesor) |

<Tip>
  Usa **Solo docentes** para material que no debe ser visible para estudiantes, como guias didacticas, solucionarios o recursos de preparacion de clases.
</Tip>

## Paso 2: Clasificación

Una vez guardado el contenido, puedes acceder al paso de Clasificación para agregar metadatos educativos.

![Paso de Clasificación](/assets/screenshots/backoffice/contenidos/editar-contenido-paso2-clasificacion.png)

### Campos de clasificación

| Campo | Descripción |
|-------|-------------|
| **País** | País al que pertenece el contenido (en sistemas multipais) |
| **Etapa** | Nivel educativo (Primaria, Secundaria, Bachillerato, etc.) |
| **Año / serie** | Grado o año escolar específico |
| **Asignaturas** | Materias o disciplinas asociadas |
| **Idioma** | Idioma del contenido |
| **Tags** | Etiquetas para facilitar la busqueda |

<Note>
  Los selectores de **Etapa**, **Año** y **Asignaturas** funcionan en cascada: primero selecciona el país, luego la etapa, y así sucesivamente.
</Note>

### Metadatos adicionales

Si tu institución tiene habilitada la funcionalidad de **Objetivos de aprendizaje**, encontrarás una sección adicional para asociar el contenido con objetivos curriculares específicos.

---

## Paso 3: Apariencia

El paso de Apariencia permite configurar aspectos visuales y de privacidad del contenido.

### Para contenidos PDF y Fixed Layout

Los contenidos de tipo PDF y Libro Digital Fixed Layout permiten subir una imagen de portada (thumbnail):

![Paso de Apariencia con thumbnail](/assets/screenshots/backoffice/contenidos/editar-contenido-paso3-apariencia-thumbnail.png)

<Steps>
  <Step title="Sube una imagen de portada">
    Haz clic en **Seleccionar fichero** para subir una imagen PNG o JPG.
  </Step>
  <Step title="Ajusta el recorte">
    Si es necesario, ajusta el área visible de la imagen.
  </Step>
</Steps>

<Tip>
  Una buena miniatura ayuda a los usuarios a identificar rápidamente el contenido. Usa imagenes representativas del tema.
</Tip>

### Configuración de privacidad

![Paso de Apariencia básico](/assets/screenshots/backoffice/contenidos/editar-contenido-paso3-apariencia.png)

| Opción | Descripción |
|--------|-------------|
| **Contenido publico** | Visible para todos los usuarios de la plataforma |
| **Contenido privado** | Solo visible para usuarios con acceso explicito |

### Opciones adicionales según tipo

Para contenidos **PDF** y **OFFICE**, aparecen opciones adicionales en el paso de Detalles:

| Opción | Descripción |
|--------|-------------|
| **Permitir descarga** | Permite a los usuarios descargar el archivo original |
| **Accesible offline** | El contenido puede verse sin conexión a internet |

---

## Paso 4: Agrupacion

El último paso permite asociar el contenido con estructuras organizativas del sistema.

![Paso de Agrupacion](/assets/screenshots/backoffice/contenidos/editar-contenido-paso4-agrupacion.png)

### Campos de agrupacion

| Campo | Descripción |
|-------|-------------|
| **Publicación** | Asocia el contenido a una publicación específica |
| **Coleccion** | Agrupa el contenido en una coleccion tematica |
| **Dependencias** | Define otros contenidos necesarios para este recurso |

<Info>
  Las **dependencias** son utiles cuando un contenido requiere otro para funcionar correctamente, por ejemplo, un ejercicio que necesita una libreria JavaScript.
</Info>

---

## Crear el contenido

<Steps>
  <Step title="Completa los campos requeridos">
    Asegúrate de completar al menos el **nombre**, **tipo de contenido** y **disponibilidad**.
  </Step>
  <Step title="Adjunta el archivo o URL">
    Según el tipo seleccionado, sube el archivo o introduce la URL.
  </Step>
  <Step title="Haz clic en Crear">
    El botón **Crear** se habilitará cuando todos los campos requeridos esten completos.
  </Step>
  <Step title="Espera la confirmacion">
    Veras una notificación de éxito y serás redirigido al detalle del contenido para completar los pasos adicionales.
  </Step>
</Steps>

<Warning>
  **No cierres la ventana** mientras se sube el archivo. Si la subida se interrumpe, deberas comenzar de nuevo.
</Warning>

## Tipos de contenido especiales

### Videos con accesibilidad

Los contenidos de tipo **Video** permiten agregar archivos de accesibilidad después de la creación:

| Archivo | Descripción |
|---------|-------------|
| **Subtitulos** | Archivos VTT con subtitulos en diferentes idiomas |
| **Audiodescripcion** | Pista de audio alternativa con descripciones |

### Audios con transcripcion

Los contenidos de tipo **Audio** permiten agregar:

| Archivo | Descripción |
|---------|-------------|
| **Transcripcion** | Archivo VTT con la transcripcion del audio |

<Note>
  Los archivos de transcripcion deben estar en formato **VTT** (WebVTT). El sistema valida automáticamente el formato.
</Note>

## Casos de uso comunes

<AccordionGroup>
  <Accordion title="Subir un video educativo">
    1. Haz clic en **Crear contenido**
    2. Escribe un nombre descriptivo (ej. "Introducción a las fracciones")
    3. Selecciona tipo **Video**
    4. Sube el archivo MP4
    5. Define disponibilidad como **Docentes y estudiantes**
    6. Haz clic en **Crear**
    7. Completa la clasificación (nivel, asignatura)
    8. Opcionalmente, agrega subtitulos
  </Accordion>
  <Accordion title="Agregar un recurso externo (enlace)">
    1. Haz clic en **Crear contenido**
    2. Escribe el nombre del recurso
    3. Selecciona tipo **Link**
    4. Introduce la URL completa
    5. Haz clic en **Crear**
    6. Clasifica el contenido para facilitar su busqueda
  </Accordion>
  <Accordion title="Crear material solo para docentes">
    1. Haz clic en **Crear contenido**
    2. Nombra el recurso (ej. "Guía del profesor - Unidad 3")
    3. Selecciona el tipo adecuado (PDF, Video, etc.)
    4. Sube el archivo
    5. En **Disponible para**, selecciona **Solo docentes**
    6. Haz clic en **Crear**
  </Accordion>
  <Accordion title="Subir un PDF descargable">
    1. Haz clic en **Crear contenido**
    2. Nombra el documento
    3. Selecciona tipo **PDF**
    4. Sube el archivo PDF
    5. Haz clic en **Crear**
    6. En el paso de Detalles, activa **Permitir descarga**
    7. Guarda los cambios
  </Accordion>
</AccordionGroup>

## Preguntas frecuentes

<AccordionGroup>
  <Accordion title="Que tamaño máximo puede tener un archivo?">
    El tamaño máximo depende de la configuración del servidor. Generalmente, los videos pueden ser de hasta 2GB y otros archivos hasta 500MB. Contacta a tu administrador para conocer los límites específicos.
  </Accordion>
  <Accordion title="Puedo editar el tipo de contenido después de crearlo?">
    No, el tipo de contenido no puede modificarse una vez creado. Si necesitas cambiar el tipo, deberas crear un nuevo contenido y eliminar el anterior.
  </Accordion>
  <Accordion title="Por que no puedo ver todos los pasos del formulario?">
    Los pasos Clasificación, Apariencia y Agrupacion solo estan disponibles después de guardar el contenido por primera vez. Completa el paso de Detalles y haz clic en **Crear**.
  </Accordion>
  <Accordion title="Como agrego subtitulos a un video?">
    Los subtitulos se agregan después de crear el contenido. Ve al detalle del video, en el paso de Detalles encontrarás la sección de subtitulos donde puedes subir archivos VTT.
  </Accordion>
  <Accordion title="Puedo crear contenido sin subir archivo?">
    Solo los tipos **Link**, **Question (Lemonade)** y **Video lesson** permiten crear contenido sin archivo. Los demas tipos requieren subir un archivo en formato compatible.
  </Accordion>
</AccordionGroup>

## Problemas comunes

<Warning>
  **Problema:** El botón Crear esta deshabilitado.

  **Solución:** Verifica que hayas completado todos los campos obligatorios: nombre, tipo de contenido y disponibilidad. Si el tipo requiere archivo, asegúrate de haberlo adjuntado.
</Warning>

<Warning>
  **Problema:** El archivo no se sube correctamente.

  **Solución:** Verifica que el formato sea compatible con el tipo seleccionado. Comprueba tu conexión a internet y el tamaño del archivo. Si el problema persiste, intenta con un navegador diferente.
</Warning>

<Warning>
  **Problema:** El contenido creado no aparece en la lista.

  **Solución:** Verifica los filtros activos en la lista de contenidos. Por defecto puede estar filtrando por algun criterio. Limpia los filtros para ver todos los contenidos.
</Warning>

<Warning>
  **Problema:** Error al validar archivo VTT de transcripcion.

  **Solución:** El archivo VTT debe comenzar con "WEBVTT" en la primera linea y seguir el formato estandar de tiempos (ej. 00:00:01.000 --> 00:00:05.000). Verifica la sintaxis del archivo.
</Warning>
